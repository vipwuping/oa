EthAddress.org is hosted on both Github pages and a dedicated server bought and provisioned by the website's maintainer.

The source code can be found on Github. For extra security, it's recommended that you download either the tarball or the zip file of the signed HTML bundle (sig & key) and run the website locally on an offline machine after verifying the signature. Alternatively, you could download the HTML bundle very quickly by just saving this page to your hard drive. (Not recommended if you plan on storing large sums of ether, though!)

Donations are always welcome at 0xa3acb4270415692cc4532a129534048364dcb1b2.

Finally, you might have noticed that there is no entropy generation step as there is at websites like BitAddress.org. This website relies on the cryptographically strong window.crypto.getRandomValues function available in modern browsers, which harvests entropy from your OS's entropy pool. If you have reason to believe that this is insecure and are willing to school me on the subject, open an issue and we'll talk it out.
